该仓库使用vite  +  storybook + vue3 搭建自己的组件库。使用Nodejs 编写 自动化脚本加速开发。

参考element-ui 代码，使用BEM css 开发 规则开发。

####  预期目标搭建自己的组件库

[X] 自动生成组件脚本[new.js]

  `npm run new:comp 组件名` 组件名最好小写驼峰

[] 使用jest组件自动化测试

[X] 使用stroybook 展示组件 

[X] 编写了第一个 Button 组件用来测试demo

#### 怎么运行项目

`yarn ` 

`yarn run dev` 运行具体项目

`yarn run storybook` 运行storybook 展示组件
